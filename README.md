# Butane

A data-driven game engine.

## Supports

| Platform  | Toolchain               | Supported             |
|:--------- |:----------------------- |:---------------------:|
| Windows   | Microsoft Visual Studio |                       |
| Windows   | MinGW/MSYS              |                       |
| Windows   | Cygwin                  |         Never         |
| Mac OSX   | GCC or Clang            |                       |
| Linux     | GCC or Clang            |                       |
| BSD       | GCC or Clang            |                       |
| Android   | GCC or Clang            |                       |
| iOS       | GCC or Clang            |                       |
| Web       | Native Client (PNaCl)   |                       |
| Web       | emscripten              |                       |
